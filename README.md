# OpenDataAT Assistant
NodeJS chatbot as a search interface to the austrian open data portals: data.gv.at and opendataportal.at

Kickstarted at the Data Pioneers Create Camp @ZAMG Vienna 2 February 2017.

## Requirements

* restify
* bluebird
* lodash
